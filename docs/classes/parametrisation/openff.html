OpenFF¶

class qubekit.parametrisation.OpenFF(*, type: typing_extensions.Literal[OpenFF] = 'OpenFF', force_field: str = 'openff_unconstrained-1.3.0.offxml')[source]¶

This class uses the OpenFFtoolkit to parametrise a molecule and load an OpenMM simulation. A serialised XML is then stored in the parameter dictionaries.
