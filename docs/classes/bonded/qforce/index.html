QForceHessianFitting¶

class qubekit.bonded.QForceHessianFitting(*, type: typing_extensions.Literal[QForceHessianFitting] = 'QForceHessianFitting', combination_rule: typing_extensions.Literal[amber] = 'amber', charge_scaling: float = 1, use_urey_bradley: bool = False)[source]¶

This class interfaces with qforce <https://github.com/selimsami/qforce> and allows users to do internal hessian fitting, this will fit all bonds, angles and rigid torsions and does not require an initial guess.

run(molecule: Ligand, **kwargs) → Ligand[source]¶

The main worker method of the class, this will take the ligand and fit the hessian using qforce and then return ligand with the optimized parameters. The ligand must contain some qm coordinates, a hessian matrix and wbo matrix.
